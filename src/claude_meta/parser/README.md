# Parser Module

> 解析器集合

## 子模块

| 解析器 | 文件 | 描述 | Bash 定位 |
|--------|------|------|-----------|
| parse_conversation | `parse_conversation.py` | 解析会话文件 | `find . -name "parse_conversation.py"` |
| parse_history | `parse_history.py` | 解析历史文件 | `find . -name "parse_history.py"` |
| parse_memory | `parse_memory.py` | 解析记忆文件 | `find . -name "parse_memory.py"` |

## 结构

```
parser/
├── README.md              # 本文件
├── parse_conversation.py  # 解析会话文件
├── parse_history.py      # 解析历史文件
└── parse_memory.py        # 解析记忆文件
```
